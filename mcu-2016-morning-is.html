Morning Camp Map Download

JDK’s Insanity

This modpack and map features boss fights with giant scorpions, monkey kings, Steve’s ghost; awesome loot like giants swords, rapid-fire bows and special armor pieces; a vertigo-inducing maze; and challenging parkour. It also has the most insane finalé ever.

Getting the Modpack

The modpack required to run this map is in Technic. Just install/open Technic, go to the modpacks tab and search for technicpack.net/modpack/morningis in the search box in the upper left. Then install that modpack.

Download the Map

Download the map here.

Install the Map

First, unzip the map file.

Next, open up the modpack in your file system. This can be done easily from within Technic:

  1. Click modpack options
  2. Click Open next to Install Folder
  3. Put the unzipped map folder into the saves folder. Here is what it looks like in Finder, for example:

That’s it! After all that you’re ready to launch Minecraft from within Technic and play the map.
